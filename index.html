My Summary
  1. The window object is the global object in a browser.
  2. Global variables are actually properties of the window object in a browser environment.
  3. Alert, confirm(), and prompt() are all methods of the window object, and open dialogs that halt the execution of the program
  4. The window.navigator object gives information about the user's browser and operating system, although it can be unreliable.
  5. The window.location object provides information about the URL of the current page.
  6. The window.history object keeps information about the pages that have been visited in the session.
  7. You can open, close, resize, and move windows (although, this doesn't mean you should!).
  8. The window.screen object provides information about the user's screen
  9. Document.write() is an archaic method of writing text to the document and should be avoided
  10. Cookies can be used to store small pieces of information between requests using the document.cookie property
  11. The window.setTimeout() method can be used to invoke a function after a set amount of time. It can be canceled using the clearTimeout() method.
  12. The window.setInterval() method can be used to repeatedly invoke a function. It can be stopped using the clearInterval() method.
  13. The window.requestAnimationFrame() method can be used to produce smooth and optimized animation by utilizing the browser's built-in graphics capabilities. It can be canceled using the cancelAnimationFrame() method.
